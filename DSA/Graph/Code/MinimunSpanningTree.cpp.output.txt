=== Minimum Spanning Tree (C++) ===

Graph (5 vertices, 7 edges)
Edges with weights: 0-1(2), 0-3(6), 1-2(3), 1-3(8), 1-4(5), 2-4(7), 3-4(1)

Minimum Spanning Tree (Kruskal's):
3 - 4: weight 1
0 - 1: weight 2
1 - 2: weight 3
1 - 4: weight 5
Total MST Weight: 11

=== Kruskal's Algorithm ===
1. Sort edges by weight
2. Use Union-Find DSU
3. For each edge (u,v):
   - If u and v not connected: add to MST
   - Union the two components
4. Continue until V-1 edges added

=== Complexity ===
Time Complexity:  O(E log E)
Space Complexity: O(V + E)
