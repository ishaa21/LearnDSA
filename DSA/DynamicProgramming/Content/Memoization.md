# Memoization

## What is this?
Top-down DP using recursion + cache.

## Why do we need it?
Easy to convert recursive code.
Faster execution.

## Algorithm Logic
Store result before returning.

## Common Mistakes
Forgetting base cases.

## Time Complexity
O(n)

## Space Complexity
O(n)

## Real Life Use
Caching computed results (e.g. API responses).

## Key Takeaways
- Recursion + memory
