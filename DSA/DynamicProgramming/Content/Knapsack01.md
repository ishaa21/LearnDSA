# Knapsack01

## What is this?
Each item can be taken once or not at all (0/1).

## Why do we need it?
Resource optimization.
Real-life packing problems.

## Algorithm Logic
Choice-based DP: either include item or exclude it.

## Common Mistakes
Wrong DP state or initial values.

## Time Complexity
O(nW)

## Space Complexity
O(nW)

## Real Life Use
Budget allocation, knapsack packing.

## Key Takeaways
- Classic DP problem
- Subset selection

## Recommended Problems
- LeetCode: 0/1 Knapsack
