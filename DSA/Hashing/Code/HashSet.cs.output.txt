=== HashSet Operations Complexity ===
Add(element):       O(1) average, O(n) worst
Contains(element):  O(1) average, O(n) worst
Remove(element):    O(1) average, O(n) worst
UnionSet():         O(m + n) where m, n = set sizes
Intersection():     O(m + n) where m, n = set sizes
Space Complexity:   O(n) where n = number of elements
