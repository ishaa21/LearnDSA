# Sorting in Arrays

## What is this?
Sorting arranges elements in order.

## Why do we need it?
- Faster searching
- Better organization

## Algorithm Logic
Depends on algorithm (Bubble, Selection, etc.)

## Common Mistakes
- Wrong comparisons
- Extra swaps

## Time Complexity
- Bubble: O(n²)
- Efficient sorts: O(n log n)

## Space Complexity
Depends on method (O(1) to O(n))

## Real Life Use
- Ranking scores
- Organizing files by size

## Key Takeaways
- Many sorting techniques
- Choice matters

## Recommended Problems
- Sort Colors (LeetCode)
- Merge Sorted Array (LeetCode)
- Bubble Sort (GFG)

## Mini Practice
### Problem: Basic Sort
Sort array in ascending order.
Input: [3, 1, 2]
Output: [1, 2, 3]
Hint: Use Arrays.sort() or a simple Bubble Sort loop.
Solution: Arrays.sort(arr); // or swap elements if arr[j] > arr[j+1]
