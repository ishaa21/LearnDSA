# Learn DSA 📚✨

**Learn DSA** is a clean, comprehensive, offline-first Flutter app designed to help students master **Data Structures and Algorithms** for placements and interviews in a structured, distraction-free way.

No login. No ads. No tracking. Just focused learning.

---

## 🚀 Key Features

### 🧠 Structured Learning
- Beginner, Intermediate, and Advanced levels.
- Topics → Subtopics → Deep dive content.
- Learn by Topic or **Learn by Pattern** (e.g., Sliding Window, Two Pointers).

### 📖 Comprehensive Topic Breakdown
Each subtopic includes:
- What is it? & Why do we need it?
- Algorithm logic & Real-life applications.
- Common mistakes & Key takeaways.
- Time & space complexity breakdown.

### 🎨 Advanced Interactive Visualizers
Unlike static tutorials, Learn DSA includes real-time, interactive algorithms visualizers for:
- Arrays, Linked Lists, Stacks, Queues.
- Sorting & Searching algorithms.
- **Advanced Trees & Graphs** (BFS, DFS, Traversals).
- **Dynamic Programming & Greedy** (Tabulation, Knapsack).
- **Hashing** (Buckets, Chaining, Collisions).
- **Heaps** (Min/Max Heap, complete tree & array synchronisation).
- **Backtracking** (N-Queens, animated step-by-step).
- **Strings** (Pointers, Substrings, Searches).

### 🧪 Hands-On Practice & Code
- Multi-language code snippets (**Python, Java, C++, C, C#**) with expected outputs.
- Multiple **Mini Practice Challenges** built directly into every topic for offline practice.

### 💼 Integrated Placement Prep Suite
A complete suite dedicated to getting you hired:
- **Aptitude Mode:** Practice Quantitative, Logical, Verbal, and Data Interpretation.
- **Resume & ATS Guide:** Built-in ATS-compliant resume builder rules, layouts, and checklists.
- **Study Plans:** Curated structured day-by-day plans (with a strict single-focus guard to keep you on track).
- **Interview Prep:** Curated behavioral and technical questions.
- **Mock Tests:** Test your knowledge under timed conditions.
- **Mistake Tracker:** Automatically log and review questions you get wrong.
- **Formula Bank:** A quick-reference cheat sheet for all DSA, Math, and Aptitude formulas.

### 🛠️ Study Tools
- **Flashcards:** Quick spaced-repetition style revision.
- **Time Complexity Guide & Placement Notes:** Cheatsheets for all major algorithms and concepts.
- **Local Leaderboard:** Track your progress against your own goals.
- **Bookmarks & Export:** Save topics and backup your progress offline.

### ⚙️ App Experience
- **Send Feedback:** Direct 1-click email integration to share thoughts.
- Font size controls and Dark/Light Mode.
- Works entirely offline.

---

## 🛠️ Tech Stack

- **Flutter** & **Dart** (UI & State Management using Provider)
- Embedded Markdown processing (`flutter_markdown`)
- Multi-language Syntax Highlighting (`flutter_highlight`)
- Static JSON Data (No backend, no cloud required)
- Local Storage (SharedPreferences)
