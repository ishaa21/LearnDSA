# 📚 Learn DSA

**Learn DSA** is a clean, offline-first Flutter app designed to help students master **Data Structures and Algorithms** for placements and interviews.

No login. No ads. No distractions. Just focused learning.

---

## 🚀 Features

- 🏠 Dashboard with progress tracking  
- 📚 Structured Topics → Subtopics → Deep dive content  
- 🧩 Pattern-Based Learning (Two Pointers, Sliding Window, DP, etc.)  
- 🧪 Practice mode with recommended problems  
- 📊 Soft progress tracking (topics, patterns, revision)  
- 🧠 Time Complexity Guide (simple & example-based)  
- ⚡ Placement Cheat Sheets for last-minute revision  
- 🔖 Bookmarks (stored locally)  
- 💾 Export/Import progress (offline backup)

---

## 🛠 Tech Stack

- Flutter  
- Dart  
- Offline JSON-based content  
- Local storage (No backend, No cloud)

---

## 🎯 App Philosophy

- ❌ No login  
- ❌ No ads  
- ❌ No distractions  
- ✅ Offline-first

- ✅ Placement-oriented  

Built for students, by students ❤️
